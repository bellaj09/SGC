the aim of this study is to test the encoding deficit hypothesis in Alzheimer disease AD using a recent method for correcting memory tests. To this end a Spanish-language adaptation of the Free and Cued Selective Reminding Test was interpreted using the Item Specific Deficit Approach ISDA which provides three indices: Encoding Deficit Index Consolidation Deficit Index and Retrieval Deficit Index. Benefits of deep encoding in Alzheimer disease. Analysis of performance on a memory task using the Item Specific Deficit Approach.
