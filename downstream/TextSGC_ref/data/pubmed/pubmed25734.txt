The implementation of an electronic teaching file using web technology is discussed in this article. A web client server model is used for a standard web browser capable of displaying joint photographic Experts Group JPEG compression images. Like other web-based teaching files this teaching file in a similar way uses a database containing information. This database section of the teaching file allows flexible database querying and viewing of pages generated by hypertext markup language HTML. Because the browser client is so flexible images types such as video and 3D representations with virtual reality markup language VRML can be displayed. Implementation of an electronic teaching file using web technology.
